# auto-branch

> A GitHub App built with [Probot](https://github.com/probot/probot).
> Simple GitHub application to create desired branches and apply branch protection rules upon repository creatory.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
