# auto-branch

> A GitHub App built with [Probot](https://github.com/probot/probot) that Simple GitHub application to create desired branches and apply branch protection rules upon repository creation.

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
